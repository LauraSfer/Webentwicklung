# Webentwicklung Projekt:
## FridgeNotes

**Erklärung**: FridgeNotes soll eine Webeseite sein, in der man sich Notizen, zu z.B. Terminen oder zu Aufgaben die man erledigen muss, machen kann. Diese sollen gespeichert und angezeigt werden. Wenn man eine Aufgabe erledigt hat, soll man die Notiz wieder löschen können.

**Gestaltung**: Wie der Name bereits verrät, soll die Webseite aussehen, als würde man Notizzettel am Kühlschrank hängen haben. Dafür soll im Hintergrung ein Ausschnitt eines Kühlschranks zu sehen sein, auf dem die Notizen erscheinen, wenn man eine neue erstellt. Unten soll es zwei Button geben, einen zum löschen und einen um eine Neue Notiz zu machen.
